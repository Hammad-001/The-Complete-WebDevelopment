1em = 16px
Text decoration may be underline, overline,line-through

Text have attributes like text-shadow, letter-spacing, text-indent.

Font can be changed as font, font-size, font-weight, font-style

Text-align has 4 properties. right, left,center, justify.

font-family can be choosen to set the type of fonts. i.e. sans-serif (without corners), serif (with corners), monospace (same width) etc.

word can be wrapped by two/three methods.like by using word-wrap property, word-break, white-space.
overflow can be set to hidden, text-overflow to elipsis to show three-dots and overflow to scroll, visible.
