Fixed means it is ignored by rest of the other elements in flow of webpage
static means default position of elements in webpage normal flow of webpage
Relative means change in position w.r.t of its original position
positioning the elements relative to the first parents that is non-static and if there is no such a parent that is non-static then positioning the elements relative to the HTML elements
z-index: Negative means below and 0 default and Positive means top
